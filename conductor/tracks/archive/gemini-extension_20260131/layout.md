# Gemini Extension Layout

## Chosen Layout

- Extension root: `adapters/gemini-extension/`
- Manifest: `adapters/gemini-extension/gemini-extension.json`
- Context file: `adapters/gemini-extension/GEMINI.md`
- Commands: `adapters/gemini-extension/commands/humanizer/humanize.toml`

## Naming

- Extension name: `humanizer-extension`
- Command group: `humanizer`
- Command name: `humanize`
